# Abinaya Udaiyar - Portfolio

A responsive portfolio website built with HTML, CSS, and JavaScript.

## Features

- Responsive design
- Smooth scrolling navigation
- Project filtering
- Skills progress bars
- Contact form
- Mobile-friendly hamburger menu

## Setup

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Customize the content with your personal information

## Customization

1. **Personal Information**: Update name, bio, contact details in `index.html`
2. **Projects**: Replace project descriptions, tech stacks, and GitHub links
3. **Skills**: Adjust skill levels and add/remove skills in the skills section
4. **Images**: Replace `placeholder.jpg` with your actual profile and project images
5. **Styling**: Modify `style.css` to change colors, fonts, etc.

## Deployment

You can deploy this static website to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome (for icons)
